### Authority layers for autonomous execution. 

I’m a systems engineer, founder, and operator with a background spanning engineering, finance, IT leadership, public-sector automation, construction manufacturing, and industrial delivery. Today I build authority, policy, and runtime layers for AI systems, alongside B2B and industrial ventures in modular construction and engineering workflows.

I'm building [KiLu](https://kilu.network/), an authority layer that adds ALLOW / REQUIRE_CONFIRM / BLOCK gates before agent actions.

Check out the [kilu-sdk](https://github.com/IkaRiche/kilu-sdk) to get started.
